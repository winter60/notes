给定一个非负整数 num。对于 0 ≤ i ≤ num 范围中的每个数字 i ，计算其二进制数中的 1 的数目并将它们作为数组返回。

示例 1:

输入: 2
输出: [0,1,1]
示例 2:

输入: 5
输出: [0,1,1,2,1,2]

来源：力扣（LeetCode）
链接：https://leetcode-cn.com/problems/counting-bits


```python
class Solution:
    def countBits(self, num: int) -> List[int]:
        ans = [0] * (num+1)
        for i in range(1,num+1):
            ans[i] = ans[i&(i-1)] + 1
        return ans
```
